# FixIt-Levenshtein

Приложение на Python для автоматической корректировки данных в Excel-файлах с использованием алгоритма расстояния Левенштейна.

## Цель проекта
Проект был разработан в рамках учебной практики. Цель — исправление опечаток и синтаксических ошибок в Excel-файлах с анкетными данными (например, студентов или клиентов).  

Программа позволяет ускорить обработку данных и уменьшить количество ручных исправлений.

## Функционал
- Ввод пользователем пути к Excel-файлу и буквы столбца, который нужно исправить.  
- Сравнение значений ячеек с эталонным списком корректных данных.  
- Автоматическая замена значений с учётом расстояния Левенштейна (поиск наиболее близкого правильного слова).  
- Графический интерфейс на Tkinter для удобного взаимодействия.  
- Работа с Excel через библиотеку `openpyxl`.

## Используемые технологии
- Python 3  
- Tkinter (GUI)  
- Openpyxl (работа с Excel)  
- Алгоритм Левенштейна (динамическое программирование)  

## Установка и запуск
1. Клонируйте репозиторий:  
```bash
git clone https://github.com/yourusername/FixIt-Levenshtein.git
2. Перейдите в папку проекта:
cd FixIt-Levenshtein
3. Установите зависимости:
pip install -r requirements.txt
4. Запустите приложение:
python src/main.py
